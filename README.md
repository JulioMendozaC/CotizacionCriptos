# CotizacionCriptos
################################################################################################################################################################################################
Instalacion:

git clone 
npm install cd Criptocotizacion 
################################################################################################################################################################################################
Tecnologias y/o Tecnicas:

React vite 
Custom hooks 
Emotion styleComponents 
Axios 
Api cryptocompare (https://min-api.cryptocompare.com/data/top/mktcapfull?limit=10&tsym=USD)

################################################################################################ 
Descripcion 
Cotizado de cripto monedas mas famosas a las monedas mas utilizadas en el mundo.

-La informacion como el nombre y precios son optenidas desde una api 
-Las monedas del mundo son un aprox de su valor real.
